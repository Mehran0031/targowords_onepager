# Targowords — one-pager

Statische landingspagina voor **targowords.com**, gehost op **GitHub Pages** (gratis).
Tijdelijke pagina voor o.a. Meta-domeinverificatie; later wordt het domein naar het
volledige Targowords-platform gewezen.

- `index.html` — de pagina (one-file, embedded CSS)
- `assets/` — logo, icon (favicon), og-afbeelding
- `CNAME` — custom domain (targowords.com)

## Meta domeinverificatie
Plak de `<meta name="facebook-domain-verification" ...>`-regel uit
Meta Business → Beveiliging → Domeinen in de `<head>` van `index.html`
(zie de gemarkeerde comment-regel), commit en push.
